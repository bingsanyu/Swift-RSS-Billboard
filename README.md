移动平台开发大作业，基于 macOS 11.0.1 和 Xcode 12.5.1 实现知乎、微博、B站热榜的RSS订阅。

使用的是 [rsshub.app](https://rsshub.app/) 域名，因此需要一点点魔法或者替换为你[自部署](https://docs.rsshub.app/zh/install)的域名。
